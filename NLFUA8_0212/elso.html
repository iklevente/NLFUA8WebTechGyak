Git és GitHub alapok

Git parancsok:

  • git init: Új Git repository inicializálása a munkakönyvtárban.
  • git clone <repository_url>: Egy távoli repository klónozása a helyi gépre.
  • git add <file>: Fájlok hozzáadása a staging area-hoz.
  • git commit -m "commit üzenet": A staging area-ban lévő változtatások commitolása.
  • git push: Lokális változtatások feltöltése a távoli repository-ba.
  • git pull: Frissítések letöltése a távoli repository-ból a lokális repository-ba.
  • git status: A repository állapotának lekérdezése.
  • git branch: Branch-ek listázása.
  • git checkout <branch_name>: Branch váltása.
  • git merge <branch_name>: Egy másik branch beolvasztása a jelenlegibe.

GitHub parancsok:

  • git remote add origin <repository_url>: Távoli repository hozzáadása a lokálishoz.
  • git push -u origin <branch_name>: Lokális változtatások feltöltése az adott branch-re a távoli repository-ba.
  • git pull origin <branch_name>: Frissítések letöltése a távoli repository-ból az adott branch-re a lokális repository-ba.
  • git clone <repository_url>: Egy távoli repository klónozása a helyi gépre.
  • git fork: Egy repository forkolása (másolása) a saját GitHub fiókodba.
  • git pull-request: Egy pull request létrehozása a változtatások felkínálásához a szülő repository-ba.
